01 - Criar um container
     docker run -it [imagem do SO] /bin/bash
     ex: docker run -it fedora /bin/bash

02 - Listar containers em execução
     docker ps

03 - Listar todos os containers 
    docker ps -a

04 - Iniciar um container
     docker start [hash do container]

05 - Vincular a um container
     docker attach [hash do container]

06 - Sair do continer sem interromper 
     CTRL + P + Q

07 - Baixar imagem mais recente do SO
     docker  pull [imagem do SO]
     ex: docker pull fedora

08 - Baixar todas as imagens do SO
     docker  pull -a [imagem do SO]
     ex: docker pull -a fedora

09 - Listar todas as imagens
     docker images

10 - Listar todas as imagens de um SO
     docker images [SO]
     ex: docker images fedora
